# Finance Tracker Backend

This backend is developed using Node.js and Express.
It handles authentication, transactions, budgeting, and data summaries.

### APIs
- POST /login
- GET /transactions/:username
- POST /transactions
- POST /budget
- GET /budget/:username
- GET /summary/category/:username
- GET /summary/monthly/:username

### Database
JSON file based storage (data.json)

### Start Server
npm install
npm start
